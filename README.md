# cpt.bouman

Personal portfolio website for **Abubakar Abdulhakeem (cpt.Bouman)** —  
a front-end developer focused on clarity, performance, and structure.

## Live Site
https://cpt-bouman.vercel.app/

## Purpose
This portfolio is designed as a **single-page front-end system** that:
- Communicates role and intent within seconds
- Uses structure and hierarchy over visual noise
- Scales cleanly as more projects are added

## Tech Stack
- **Next.js (App Router)** — structure & routing
- **React** — component-based UI
- **TypeScript** — type safety
- **Tailwind CSS** — predictable, utility-first styling
- **Vercel** — deployment & hosting

## Key Features
- Single-page navigation with active section tracking
- Responsive layout (mobile-first)
- Minimal UI focused on readability and UX
- Reusable project section for future scalability
- Downloadable CV via public assets

## Architecture Notes
- Static generation for speed and SEO
- Intersection Observer for navigation state
- Clear separation of sections (Hero, Projects, About, Contact)
- Components designed for reuse and clarity
